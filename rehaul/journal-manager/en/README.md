# Learning OJS 3.5 for Journal Managers: Configure Your Journal

Open Journal Systems (OJS) is an open source solution to managing and publishing scholarly journals online. OJS is a highly flexible editor-operated journal management and publishing system that can be downloaded for free and installed on a local Web server.

This guide is a comprehensive manual covering the various functions and configuration options of OJS version 3.5, released in March 2025. It contains information on everything Journal Managers or Journal Editors will need to use the software to publish and manage content. It provides detailed instructions with screenshots for configuring all aspects of OJS, including entering key journal details, preparing the journal site and its appearance, preparing policy and guidelines for receiving submissions, managing users and communication with users, and configuring access to the site and its content. By following this guide, users will be able to successfully prepare OJS to receive, publish, and display academic articles on OJS.
